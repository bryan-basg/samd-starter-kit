# Dependencias del sitio de documentación (MkDocs Material).
#
# Uso:
#   pip install -r requirements-docs.txt

#
# mkdocs-material arrastra transitivamente mkdocs, pymdown-extensions y pygments,
# que son todo lo que el sitio necesita: la búsqueda y el render de Mermaid son
# nativos de Material (pymdownx.superfences), sin plugins externos ni Node.
# Pin por versión MAYOR para builds reproducibles sin congelar parches de seguridad.
